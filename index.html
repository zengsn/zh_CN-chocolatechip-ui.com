版本 1.0

对！就是这个版本！几个月来，我努力优化 ChocolateChip-UI，尽力使之成为最优秀的 Webkit 平台移动 Web 应用程序开发框架。这个版本已经针对 iOS 5 进行了优化。

jQuery Port 已经发布

现在 ChocolateChip-UI 已经支持 jQuery。

ChocolateChip-UI 支持 Zepto

ChocolateChip-UI 现在支持微框架 Zepto。

ChocolateChip 现在变成 ChUI（发音是“chewy”）

ChocolateChip-UI 框架内置了布局和小部件功能，可用于快速开发移动 Web 应用程序。有时候，应用程序不支持脚本。开发者只需要创建标记代码，ChocolateChip-UI 会帮您添加原本用脚本实现的功能，如导航列表、返回等。另外，我们有时候需要通过 JavaScript 初始化小部件。这样会增加应用程序获取数据的开发难度。因为我们目标是使程序尽可能轻量，ChocolateChip-UI 在创建小部件和控件时，完全不需要使用图像，所有元素都由 WAML 和 CSS 创建。ChocolateChip-UI 使用矢量图像 SVG 创建按钮的图标。这意味着，Web 应用程序不受分辨率的限制，绝对简洁！欢迎使用最新的 ChUI ChocolateChip。

ChocolateChip-UI 已支持 jQuery

ChocolateChip-UI 已经支持 jQuery。如果您正在使用 jQuery，那么您可以使用这个版本。请从 Github 下载框架，然后通过例子学习框架的使用方法。如果您之前使用了普通版本的 ChocolateChip-UI，并且想转换到 jQuery 版本，那么请仔细学习例子。由于 ChocolateChip 和 jQuery 底层有一定的区别，所以有一些控件回调函数的数据访问方法已经发生变化。请从 Github 下载 jQuery 版 ChocolateChip-UI。

ChocolateChip-UI 已支持 Zepto

ChocolateChip-UI 已经支持 JavaScript 微框架 Zepto。如果您喜欢一个超小且专业的 JavaScript 框架进行移动开发，但是又需要一些前端外观体验，那么这个版本正好适合您使用。ChocolateChip-UI 的 Zepto 版本是一种强大但轻量的 Web 应用程序开发框架。请从 Github 下载 Zepto 版 ChocolateChip-UI。

温馨提示

ChocolateChip 面向安装了基于 Webkit 的浏览器的移动设备。因此，在开发过程中，您可以在桌面电脑上使用任何基于 Webkit 的浏览器，其中包括 Google Chrome 和 Apple Safari。ChocolateChip-UI 提供了一些桌面兼容性，以方便测试。您也可以使用模拟器。最后，您必须在真实设备上测试，毕竟桌面浏览器与模拟器之间存在一些细微差别。
